# CCMA Cat Clinic Tycoon

A React + Vite browser game project for CCMA study progression with a tycoon-style clinic map.

## Project Structure

- `index.html`
- `package.json`
- `src/main.jsx`
- `src/App.jsx`
- `src/styles.css`
- `src/data/questions.js`
- `src/data/upgrades.js`
- `src/utils/gameLogic.js`

## Notes

- This repository includes the full source scaffold and game logic/data files.
- Study content is intended as supplemental practice and should be used with official NHA CCMA materials.
